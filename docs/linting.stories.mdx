# Baklava testing

> This guide will help contributors about how to lint Grace.

### Brief
- To lint, we are using `tsc`, `eslint`, `prettier` and `stylelint`.
    - To get more information, you can find our configurations at [tsconfig.json](https://github.com/Trendyol/grace/blob/next/tsconfig.json), [.prettier.json](https://github.com/Trendyol/grace/blob/next/.prettier.json), [.eslintrc.json](https://github.com/Trendyol/grace/blob/next/.eslintrc.json) and [stylelintrc.json](https://github.com/Trendyol/grace/blob/next/stylelintrc.json).
- To run linters, you can simply type `npm run lint` and run.
