# Pagination

The Pagination component enables the user to select a specific page from a range of pages. Number of content per page can also be selected by the user


## Simple Pagination
Use a simple pagination for navigating to the next or previous page.

<Canvas>
<Story name="Basic Usage" args={{currentPage:1,totalResults:200,resultPerPage:5 }}>
    {SimplePagination.bind({})}
  </Story>
</Canvas>

## Jumper
You can use the jumper to directly go to a specific page. The jumper label can be customized.

<Canvas>
<Story name="Jumper" args={{currentPage:1,totalResults:200,resultPerPage:5,hasJumper:true }}>
    {Jumper.bind({})}
  </Story>
</Canvas>
