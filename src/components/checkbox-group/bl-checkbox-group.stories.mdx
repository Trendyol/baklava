# Checkbox Group
Checkbox groups organizes the multiple checkbox states and parent checkbox shows the result.

### Usage
* Parent checkbox automatically shows up and it can has indeterminate state.
* Child checkboxes should be written as Checkbox Group's slot
* Parent checkbox is checked when all child checkboxes are checked
* Parent checkbox is unchecked when all child checkboxes are unchecked
* Otherwise parent checkbox has indeterminate state

## Basic usage
You can show label by just using label attribute.
