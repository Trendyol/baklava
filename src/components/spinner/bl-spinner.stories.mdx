# Spinner

The `<bl-spinner>` component is used to indicate loading states. It can be customized with different sizes.

## Stand-alone

<Canvas>
  <Story name="Default Spinner" args={{ size: 'medium'}}>
    {(args) => html`<bl-spinner size=${ifDefined(args.size)}></bl-spinner>`}
  </Story>
</Canvas>

## Spinner Sizes

<Canvas>
  <Story name="Spinner Sizes">
    {() => html`
      <div>
        <bl-spinner size="xxsmall"></bl-spinner>
        <bl-spinner size="xsmall"></bl-spinner>
        <bl-spinner size="small"></bl-spinner>
        <bl-spinner size="medium"></bl-spinner>
        <bl-spinner size="large"></bl-spinner>
        <bl-spinner size="xlarge"></bl-spinner>
        <bl-spinner size="xxlarge"></bl-spinner>
      </div>
    `}
